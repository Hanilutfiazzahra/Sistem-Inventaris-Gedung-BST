# Sistem Inventaris Gedung (BST)

Tugas Besar mata kuliah Struktur Data.  
Program ini merupakan implementasi Binary Search Tree (BST) untuk mengelola data inventaris pada suatu gedung.

## Deskripsi Singkat
Sistem menyimpan data ruangan berdasarkan ID ruangan sebagai key BST.  
Setiap ruangan memiliki daftar barang inventaris yang tersimpan dalam struktur data array.

## Fitur Program
- Menambahkan data ruangan ke dalam BST
- Menambahkan data barang pada ruangan tertentu
- Menampilkan data inventaris menggunakan traversal BST
  - Inorder
  - Preorder
  - Postorder
- Pencarian data ruangan berdasarkan ID

## Struktur File
- `main.cpp` – Program utama
- `Tree.h` – Definisi struktur data BST dan fungsi
- `Tree.cpp` – Implementasi fungsi BST

## Cara Menjalankan
1. Compile seluruh file `.cpp`
2. Jalankan program melalui terminal / IDE
3. Masukkan data sesuai menu yang tersedia
